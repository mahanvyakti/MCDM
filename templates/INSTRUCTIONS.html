INSTRUCTIONS

A) SWARA

  1. Click on ‘MCDM Simulator’ on the Navigation Bar.

  2. Click on ‘SWARA’ Button under the MCDM Techniques.

  3. A new interface of SWARA Calculator opens up. Enter the names of experts separated by comma in the Expert Input Box.

  4. Click on ‘Main Criteria’ and enter all the main criterion separated by comma.

  5. Click on ‘Sub-Criteria’ and enter the Sub-criterion for each main criteria as the input box prompts.

  6. Click on ‘Main Criteria Importance Values’ and enter the Importance Values for each Main Criteria by each expert. The Importance Values must be between 1-7.

  7. Click on ‘Sub-Criteria Importance Values’ and enter the Importance Values for each Sub-Criteria by each expert.

  8. Click on ‘Edit Criteria Details’ if you want to edit any of the above information else Click ‘ Add Comparative Significance’

  9. Click on ‘Main Criteria Comparative Significance Values’.The Comparative Significance Value for the first Main criteria or Sub-Criteria after sorting is by default given as 0.

  10. Add the subsequent comparative significance values for the main criterion by each expert. These values must strictly be between 0-1.

  11. Click on ‘Sub-Criteria Comparative Significance Values’ and add the subsequent comparative significance values for the sub-criterion by each expert.

  12. Click on ‘Get Results’ to get all the Intermediate as well as Final Results.


B) TOPSIS

  1. Click on ‘MCDM Simulator’ on the Navigation Bar.

  2. Click on ‘TOPSIS’ Button under the MCDM Techniques.

  3. A new interface of TOPSIS Calculator opens up.

  4. Enter all the names of criteria separated by comma in the Criteria Input Box.

  5. Similarly, enter all the Alternatives separated by comma in the Alternatives Input Box

  6. Click on ‘Weights’ and enter the weights for all the Criteria as prompted.

  7. Click on ‘Beneficiality’ and select all the criteria that are beneficial.

  8. Click on ‘Edit Criteria and Alternatives’ if you want to edit any of the above information else Click ‘ Add Criteria Values’

  9. Click on ‘Criteria Values’ and enter all the criteria values for each alternative in the corresponding boxes.

  10. Click on ‘Get Results’ to get all the Intermediate as well as Final Results.
